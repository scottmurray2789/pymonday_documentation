Teams

get_teams()

get_teams()

Description: Get Names and IDs of all teams.

Arguments: None

Returns: Dictionary of IDs and Names. Return None on API call failure.

Example:

teams = monday.get_teams()

get_team_members()

get_team_members(team_ids=List)

Description: Get User ID, Name & Email of team members.

Arguments:

team_ids: UUIDs of the Teams to be retrieved. List of integers.

Returns: Dictionary of Teams and member data. Return None on API call failure.

Example:

team_data = monday.get_team_members(team_ids=[12345, 54321])
